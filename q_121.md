# С++ 
## Общие сведения и отношение с С

**С++** это компилируемый, статически типизированный язык программирования.

По сравнению с С, С++ включает в себя:

- Тип **bool**
- Ссылки
- Пространство имён
- Классы и всё с ними связанное
- Шаблоны
- Перегрузка в т.ч. операторов

И многое другое

БОльшая часть кода С будет справедлива для С++

**НО** С++ не является надмножеством С и не включает его в себя.
